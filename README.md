# PlayVoice

一款  更现代 更便携 操作更简单  的 游戏内麦克风音频播放工具，将欢乐的语音播放到你的队麦。

优势
创意工坊：  内置创意工坊查看与上传，发现更多好玩的音频，或将你收集好的音频共享出去，独乐乐不如众乐乐。

云存档 + 小而精：  程序占用空间小，出门外在可以随时在steam下载此程序，并配合云存档让你随时随地使用你的预设。

热键：  为每一个音频独立设置一个热键，在游戏内快速播放，不再需要频繁切换界面。

按键适配：  为常见的多人组队游戏适配按键，播放音频会自动按下游戏麦克风按键，将更多的专注留给游戏。

音量设置：  为每一个音频设置单独的音量，保证它们既不会干扰队友，又不会使队友视而不见听而不闻。

免费使用：  如果你喜欢此产品，我们理解您暂时囊中羞涩，可以前往我们的 GitHub 网站免费获取，免费获取的产品不支持创意工坊与云存档功能。
